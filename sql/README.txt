## Database Setup Instructions
1. Open PHPMyAdmin and create a new database. Call it 'acmearts'
2. Navigate to the **Import** tab in PHPMyAdmin.
3. Select the `database.sql` file from the project folder.
4. Click **Go** to import the database structure and sample data.
OR
5. Run the database.sql as a query in the acmearts database and it will import. 
6. Go to the import_images.php in your URL to import the images into BLOBS.
